npm 和 npx 的区别

CrabSAMA小于 1 分钟node.jsnode.js

npm 是一个包管理器，npx 类似于一个运行需要全局安装的 npm 包的快捷指令，会在需要时下载对应的包，用完即删除。在 npm 的基础之上，npx 让 npm 包中的命令行工具和其他可执行文件在使用上变得更加简单。
